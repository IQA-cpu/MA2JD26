# Management Accounting 1 (ACCT 221) — Course Hub (all-HTML)

Every page is a single self-contained `.html` file — photos, music, scripts and image-slot photos are all embedded. No other files are needed.

## Deploy to GitHub Pages
1. Create a repository and upload every `.html` file (keep flat).
2. Settings → Pages → Deploy from branch → `main` / root.
3. Live at `https://<username>.github.io/<repo>/`. Entry point: `index.html`.

## Notes
- The two hero background videos (Welcome, Corridor) stream from an external URL and need internet; everything else is embedded.
- Fonts load from Google Fonts (needs internet); the design falls back to system fonts offline.
